How the large-data flow works

This page is for demonstrations. The actual dashboard opens from the button above, where users upload their own local files and interact with the filtered pharma dashboards.

1

Upload locally

Users choose Excel, CSV, TSV, SQL, or JSON files. The app keeps the file in the browser session.

2

Parse off the UI thread

Large CSV, TSV, SQL, and XLSX work is handled in a Web Worker so the dashboard remains responsive.

3

Map pharma fields

MarketLens IQ detects Brand, Company, Therapy, Molecule, Sales, Units, MAT, Month, and Pack columns.

4

Filter and analyze

Dashboard selections use optimized filter matching so views refresh from the loaded dataset quickly.

Best format for 500MB files

  • CSV, TSV, and SQL dumps are the fastest large-file options.
  • XLSX is supported, but speed depends on browser memory and workbook structure.
  • For very large Excel workbooks, exporting the needed sheet as CSV is recommended.

What to show during a demo
