# Contributing To MarketLens IQ

MarketLens IQ is owned and maintained by `@SiDjr18`.

## Change Control

- Do not commit directly to the main project unless you are the repository owner.
- Suggested changes should be sent through a GitHub pull request.
- The owner reviews and decides whether to merge any pull request.
- Do not upload IMS, IQVIA, IPM, client, patient, doctor, prescription, or confidential business data to this repository.

## Privacy

This app is designed as a static browser app. Uploaded files are processed in the user's own browser and are not sent to a MarketLens IQ server.

## Free Deployment

The default deployment target is GitHub Pages from the `main` branch. This keeps hosting free for a public static site under GitHub's normal free Pages limits.

## Important GitHub Limitation

GitHub does not notify the owner every time a public repository is viewed, cloned, or downloaded. To monitor interest, use public signals such as stars, forks, issues, pull requests, and GitHub traffic analytics when available in the repository Insights tab.
